# Privacy Policy - Elixir: Summer Escape

**Last Updated:** January 2025  
**Effective Date:** January 2025

## Introduction

Elixir: Summer Escape ("the Bot") is a Discord gaming bot that provides an interactive adventure experience. This Privacy Policy explains what data we collect, how we use it, and your rights regarding your data.

---

## 1. Data We Collect

### 1.1 User Information
- **Discord User ID**: Your unique Discord identifier
- **Username**: Your Discord display name (for leaderboards and game displays)
- **Server/Guild ID**: ID of Discord servers where you use the bot

### 1.2 Gameplay Data
- **Game Progress**: Battle pass level, XP, achievements
- **Inventory**: Fish caught, collectibles, equipment
- **Currency**: Seashells, sun tokens, prestige points
- **Statistics**: Fish caught count, islands visited, challenges completed
- **Trading History**: Record of completed trades (items exchanged, not chat logs)
- **Tournament Participation**: Scores and rankings in competitions

### 1.3 Technical Data
- **Command Usage**: Which commands are used (for analytics and improvements)
- **Timestamps**: When actions occur (for cooldowns and time-based features)
- **Server Settings**: Custom prefix settings per server

---

## 2. Data We DO NOT Collect or Store

We want to be crystal clear about what we **DO NOT** do:

❌ **Message Content**: We process message content in real-time to detect commands (like "cast" in tournaments), but we DO NOT store, log, or archive message content  
❌ **Chat Logs**: We do not keep records of conversations  
❌ **Personal Information**: No email, phone number, real name, or address  
❌ **Voice Data**: We do not access voice channels  
❌ **Direct Messages**: We do not read or store DMs  
❌ **Activity Outside Bot**: We don't track what you do elsewhere on Discord

---

## 3. How We Use Data

### 3.1 Primary Uses
- **Game Functionality**: Store your progress, inventory, and statistics
- **Command Processing**: Parse commands to execute game actions (real-time only)
- **Leaderboards**: Display rankings and competitive stats
- **Anti-Abuse**: Detect spam, cheating, and exploits to maintain fair play
- **Features**: Enable trading, tournaments, challenges, and progression

### 3.2 Message Content Processing
**Important**: Message Content Intent is used ONLY for:
- Detecting game commands in chat (e.g., "cast" during tournaments)
- Processing text responses in interactive exploration events
- Parsing trade offers in chat
- Identifying spam/abuse patterns

**Message content is:**
- ✅ Processed in real-time (in bot memory)
- ✅ Immediately discarded after processing
- ❌ NEVER stored in our database
- ❌ NEVER logged to files
- ❌ NEVER used for analytics or tracking
- ❌ NEVER shared with anyone

### 3.3 Analytics & Improvements
- Command usage statistics (which features are popular)
- Error tracking to fix bugs
- Performance monitoring

**Note**: Analytics are aggregated and anonymous. We track "1000 people used fishing today," not "User123 used fishing."

## 6. Your Rights & Choices

### 6.1 Access Your Data
Request a copy of your stored data by contacting us at daisukiedev@gmail.com

### 6.2 Delete Your Data
You can request account deletion:
- **Method 1**: Use the `/delete-account` command (if implemented)
- **Method 2**: Contact us at daisukiedev@gmail.com
- **Method 3**: Remove the bot from all your servers (auto-purge after 90 days)

**Note**: Deletion is permanent and cannot be undone. All progress will be lost.

### 6.3 Opt-Out Limitations
You cannot opt-out of message content processing while using interactive features (tournaments, events). This is because:
- Features require processing all participants' commands equally
- We don't store content, only process it
- Alternative: Simply don't use the bot if you prefer not to have commands processed

### 6.4 Server Owner Rights
Server administrators can:
- Remove the bot to stop data collection for that server
- Request data deletion for all server members
- Change server-specific settings
